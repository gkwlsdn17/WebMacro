인터파크
1. 등급이 있는 예매의 경우
config에서 bookInfo에
grade는 VIP, R석과 같은 등급을 제어하고 싶을 때 사용
zone은 예매하기 눌러서 VIP눌렀을 때 001구역 처럼 세부 구역이 나오면 해당부분을 제어하고 싶을 때 사용
order는 실제 가구역, 나구역 처럼 포스터에서 볼 수 있는 구역이름으로 제한하고 싶을 때 사용
colOrder는 열 번호 제어하고 싶을 때 사용
function에서 special_area를 Y로 해야 위의 기능 사용 가능

2. 전석으로만 되어있는 예매의 경우
zone은 사용하지 않음
order로만 원하는 부분 제어


고쳐야할 오류요소
1. 등급으로 "전석" 하나만 있을때 특정구역 지정 안할때 최초에는 모든 구역을 다 잘 도는데, 두번째부터 등급 클릭이 안되는지 진행이 안됨
등급이 한 개일 때 오류가 나는것인지 확인 필요하고 수정 필요함
